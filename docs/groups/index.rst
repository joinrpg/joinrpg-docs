Группы
=======

Группы персонажей используются при формировании сетки ролей и заменяют привычные «локации». Они могут вкладываться друг в друга любым образом, персонаж может относиться к любому числу групп. При создании проекта автоматически создается группа «все роли», включающая всех персонажей игры.

Замена привычных «локаций» на «группы» — одно из ключевых отличий joinrpg.ru от allrpg.info: настройки как для сетки ролей в целом, так и для отдельных персонажей теперь более гибкие и удобные для мастеров.
